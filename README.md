# Tarea1-PaginaUmad
Esta pagina web cuenta con la estructura base  (&lt;!DOCTYPE html>, &lt;html>, &lt;head>, &lt;body>)
Encabezado <header>.
Navegación <nav>.
Una sección de la página (por ejemplo, el banner de inicio) usando <section>.
Una sección adicional usando <section> que contenga al menos dos artículos (<article>) dentro.
Si la página que elegiste cuenta con una barra lateral, incluye un <aside>.
Pie de página <footer>.
